HTML Theory In Mind

Our Basic Building Blocks.

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It defines the meaning and structure of web content.

Simplicity

Avoid over- nesting, Eliminate Unnecessary Divs. Keep CSS & JS external.

Readable

Use semantic elements such as "header", "nav", "section", "footer" etc. to devide your page into logical sections. Give classes names based on what they do rather than how they look like. keep a logical Ordering of the elements and do a proper indentation.

Consistency

Use either single or double quotes, but stay consistent with whatever you choose. Use lowercase for elements, attributes, and values, as lowercase is the preferred convention.

CSS Theory In Mind

A major tool for creating beautiful websites.

CSS (Cascading Style Sheets) is used for describing the presentation of a document written in HTML and designed to enable the separation of presentation from content, including layout, colors, and fonts. This separation helps control the layout of multiple web pages at once.

Maintenance

Overall prefer usage of class over id selectors- because what appears unique now may not remain so in the future. I try to limit each element to a single selector whenever feasible, while defining some class selectors for components that are used frequently to promote consistent styling.

Separation

Complete separation of css from html. CSS splitted to different files where each match a different page and one is global. In each File I used comments to separate styles of different sections, ensuring better organization and readability of the styles.

Responsiveness

The website design should be suitable for different window sizes as well as for different devices such as computer, mobile phone, tablet, etc.

Design Theory In Mind

A system of ideas that explains how and why design works.

It provides designers with an understanding of the elements and principles of visual communication, as well as how they can be applied to solve problems through design thinking. Design theory is the asking and answering of the question “Why am I designing it this way?”.

Color Theory

Body of practical guidance to color mixing and the visual effects of a specific color combination.

Typography Theory

The art of arranging a message in a readable and aesthetically pleasing composition.

Layout Theory

The overall appearance and relationships between the graphic elements to achieve a smooth flow of message and eye movement for maximum effectiveness.

Logo Design Theory

Logo design is all about creating the perfect visual brand mark for a company.
